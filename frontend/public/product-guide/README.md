# Recursos visuais do manual (`/guide`)

Coloque aqui ficheiros estáticos servidos pelo Vite em **`/product-guide/<nome>`** (respeitando `import.meta.env.BASE_URL` na app).

Sugestão de nomes (opcional):

| Ficheiro | Uso sugerido |
|----------|----------------|
| `cover.png` | Capa ou vista geral (mostrada no topo da página `/guide` se existir) |
| `dashboard.png` | Painel principal |
| `investigations.png` | Lista de investigações |
| `settings-integrations.png` | Ecrã de integrações |

Formatos: PNG ou WebP, largura recomendada ≤ 1200px para performance.

Não versionar ficheiros muito pesados sem necessidade; comprimir imagens antes do commit.
